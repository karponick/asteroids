# Asteroids Knockoff
This game was made to try to replicate the classic arcade game Asteroids using the Godot engine.  

## Controls
Key | Action
-|-
← → | Rotate
↑ | Thrust
F | Fire
S | Pause
R | Restart

## Tools
- Godot
- gdscript
